# 公众号文章自动生成系统

基于 AI 的公众号文章自动化编写工具，支持从选题到生成完整文章的完整流程。

## 功能特点

- ✨ **AI 智能生成**：使用通义千问/AI 生成高质量的公众号文章
- 📝 **多种文章类型**：支持技术教程、资讯解读、观点评论、产品推广等类型
- 🎯 **三步完成写作**：输入主题 → 确认大纲 → 生成全文
- 📚 **历史记录管理**：自动保存生成的文章，方便查阅和管理
- 📥 **多格式导出**：支持 Markdown、HTML（微信公众号格式）导出
- 🔥 **热点追踪**：实时获取微博、知乎、百度、抖音、36 氪热搜榜
- ⏰ **定时任务**：支持定时自动生成文章、定时发布
- 📱 **公众号同步**：一键同步到草稿箱，支持正式发布
- 🎨 **12 种排版主题**：默认蓝、清新绿、活力橙、优雅紫、中国红、商务黑、文艺青、甜美粉、科技蓝、复古棕、高端金、极简灰
- 📧 **邮件通知**：任务执行完成后发送邮件通知
- 📁 **文件管理**：在线浏览、编辑、批量删除 Markdown 文件
- 🐳 **Docker 部署**：一键容器化部署，简单方便

## 快速开始

### 方法一：本地运行

1. **克隆项目**
```bash
cd wechat_auto_writer
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **配置环境变量**
```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，填入你的通义千问 API Key
# DASHSCOPE_API_KEY=your_api_key_here
```

> 通义千问 API Key 获取：访问 https://dashscope.console.aliyun.com

4. **启动应用**
```bash
streamlit run app/main.py
```

5. **访问应用**
打开浏览器访问：http://localhost:8501

### 方法二：Docker 部署

1. **配置环境变量**
```bash
cp .env.example .env
# 编辑 .env 文件，填入 API Key
```

2. **启动容器**
```bash
docker-compose up -d
```

3. **访问应用**
打开浏览器访问：http://localhost:8501

## 使用指南

### 1. 配置 API Key

首次使用需要在侧边栏配置 AI API Key：
- 访问对应平台的控制台获取 API Key
- 在侧边栏输入并保存

### 2. 生成文章

**第一步：输入主题**
- 填写文章主题/关键词
- 选择文章类型（通用、技术教程、资讯解读、观点评论、产品推广）
- 可选：填写特殊要求（字数限制、风格要求等）

**第二步：确认大纲**
- 预览 AI 生成的文章大纲
- 可编辑标题、副标题和章节内容
- 查看封面图建议

**第三步：预览文章**
- 查看完整文章内容
- 支持 Markdown 实时预览
- 导出为 Markdown 或 HTML 格式（12 种排版主题可选）
- 保存到历史记录

### 3. 历史记录

- 在侧边栏切换"历史记录"页面
- 查看所有已保存的文章
- 支持复制、下载、删除操作

### 4. 热点追踪

- 在侧边栏切换"热点追踪"页面
- 支持平台：微博、知乎、百度、抖音、36 氪
- 一键生成热点文章

### 5. 定时任务

- 在侧边栏切换"定时任务"页面
- 支持每天/每周/每小时定时执行
- 支持固定主题或根据热点生成
- 支持自动同步到草稿箱/正式发布
- 支持邮件通知

### 6. 文件管理

- 在侧边栏切换"文件管理"页面
- 浏览 output 目录所有 Markdown 文件
- 在线预览和编辑
- 批量删除操作
- 一键同步到公众号

## 高级配置

### 多平台 AI 支持

支持多个 AI 平台，在侧边栏切换：
- **通义千问**: https://dashscope.console.aliyun.com
- **OpenAI**: https://api.openai.com
- **DeepSeek**: https://api.deepseek.com
- **智谱 AI**: https://open.bigmodel.cn
- **自定义平台**: 支持任意 OpenAI 兼容 API

### 微信公众号发布（可选）

如需直接发布到微信公众号，需要在侧边栏配置：
- 公众号 AppID
- 公众号 AppSecret

> 注意：需要已认证的服务号才能使用发布功能

### 邮件通知配置（可选）

如需定时任务完成后发送邮件通知，需要在侧边栏配置：
- SMTP 服务器地址
- SMTP 端口
- 发件人邮箱
- 邮箱密码/授权码
- 收件人邮箱列表

## 常见问题

### 1. API Key 无效
请确认：
- API Key 复制正确，没有多余空格
- 账户有足够的额度
- 网络连接正常

### 2. 生成速度慢
- 首次生成可能需要 1-2 分钟
- 确保网络连接稳定
- 检查 API 调用配额

### 3. Docker 启动失败
```bash
# 查看日志
docker-compose logs

# 重新构建
docker-compose up -d --build
```
