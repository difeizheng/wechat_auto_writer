# v2.4 版本记忆 (2026-03-24)

## 新增功能速查

### 1. 流式输出支持
- **文件**: `app/generator.py`
- **改动**: 为 `generate_outline()`, `write_article()`, `modify_article()` 添加 `stream` 和 `stream_callback` 参数
- **回调签名**: `stream_callback(delta: str, done: bool)`
- **使用示例**:
```python
async def stream_callback(delta, done):
    if delta:
        print(delta, end='')
    if done:
        print("\n生成完成!")

await generator.generate_outline(topic="AI 发展", stream=True, stream_callback=stream_callback)
```

### 2. 多轮交互修改文章
- **文件**: `app/main.py` - `step3_preview_article()`, `modify_article()`
- **快速修改选项**: 优化标题、增加案例、简化语言、增强互动性、添加总结、扩展开头
- **Session State**:
  - `modification_history`: 修改历史记录列表
  - `current_article_content`: 当前文章内容
- **操作流程**: 输入修改要求 → 执行修改 (流式显示) → 选择使用此版本/继续修改/放弃修改

### 3. Docker 部署优化
- **Dockerfile**: 多阶段构建、非 root 用户、健康检查
- **docker-compose.yml**: 资源限制 (CPU 2.0, 内存 2G)、健康检查、自定义网络
- **启动命令**:
  ```bash
  # Linux/Mac
  ./docker-start.sh

  # Windows
  docker-start.bat

  # Makefile
  make start / make stop / make logs
  ```

### 4. 自动化测试框架
- **测试工具**: pytest + pytest-asyncio + pytest-cov
- **测试目录**: `tests/`
  - `conftest.py`: 共享夹具和测试配置
  - `test_generator.py`: 13 个单元测试
  - `test_integration.py`: 14 个集成测试
- **运行测试**:
  ```bash
  pytest -v                    # 普通测试
  pytest --cov=app            # 带覆盖率
  ./run_tests.bat             # Windows 脚本
  ```
- **当前覆盖率**: 总计 17%, generator.py 97%, models.py 99%
